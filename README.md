# EX Automation

Projeto de automação para leitura de arquivos PDF e preenchimento de planilha Excel.

## Funcionalidades
- Leitura de arquivos em pastas por empresa
- Extração de nome e votos
- Preenchimento automático em planilha

## Como usar
1. Rodar o script
2. Informar o caminho da pasta
3. Informar o caminho da planilha
